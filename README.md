# EE 234 — Microprocessors (Spring 2026)
# Gus Ballman

Work from WSU’s EE 234 ARM‑based microprocessors course. Projects move from basic ARM assembly to C programs that control [Real Digital](https://www.realdigital.org/hardware/blackboard) Blackboard peripherals.

---

## Focus
- ARM assembly fundamentals  
- Bare‑metal C on the Cortex‑A9  
- Memory‑mapped I/O for LEDs, switches, timers, and sensors  
- Intro ARM interrupts  
- Building low‑level drivers and simple embedded systems
